너는 20년 경력의 시니어 웹 개발자야.
나는 지금 토스 증권 open api를 사용해서 자신이 가지고 있는 주식을 직접 입력하면 해당 주식을 실시간 보여주는 웹을 만드려고 해.

기술 스텍
frontend: react vite
backend: node.js express.js
배포: oracle cloud free tier

웹의 흐름
사용자 로그인
    |
'어서오세요 ***님'
    |
메인 페이지

필요한 기능: 읽기, 추가, 삭제, 수정, 주식 이름 검색, api로 실시간 주식 현황 가져오기, 그래프로 보여주기

로그인 방식은 추천이 필요해.
'어서오세요 ***님'에서 자연스럽게 '어세오세요'는 흐려지면서 사라지고 '***님'이 우즉 상단으로 올라가면서 '***님의 주식 현황'이라는 타이틀로 변화하면 좋겠어.
'***님의 주식 현황'이 타이틀은 메인 페이지의 우측 상단에 위치 하겠지.

메인 페이지에서는 기본적으로 테이블 형식으로 하면 좋을 것 같은데 더 좋은 방식이 있다면 추천해줘.
추가한 주식이 아직 없다면 정중앙에 '아직 보유한 주식이 없어요!' 텍스트와 그 바로 아래 '추가하기' 버튼을 만들어줘.
주식을 추가한 후에는 프론트 엔드에서 리스트와 데이터베이스에 저장하고 실시간 api를 불러와서 계속 업데이트 해주는 방식이면 좋겠는데 더 효율적인 방식을 추천해줘.
